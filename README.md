## 📋 What Goes Where

- **Strategy & planning** → `docs/`
- **Print designs (NFC cards, table tents, stickers)** → `kit-designs/`
- **Training guides & compliance** → `training/`
- **Brand assets (logos, photos)** → `branding/`
- **App source code** → `tapcard/`, `content-creator/`, etc.
- **Prompts for Manus/Lovable** → `prompts/`
